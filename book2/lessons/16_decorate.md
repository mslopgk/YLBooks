# 16차시 · 게임 꾸미기

!!! note "이번 차시에 하는 일"
    - 게임 전체의 **색깔과 분위기**를 내 마음대로 바꿔 봅니다
    - **시작화면의 제목·설명 문구**를 새로 써 봅니다
    - 노트를 잘 맞췄을 때 **반짝이는 효과** 등을 더해 봅니다

> ⏱️ 걸리는 시간: 약 25분 · 🧰 준비물: 11~15차시에서 만든 게임

---

## 왜 이걸 하나요?

지금까지 만든 게임은 이미 잘 작동합니다. 노트가 떨어지고, 키를 누르면 판정이 나고, 점수도 쌓입니다. 뼈대는 다 세운 집이나 마찬가지죠. 이번 차시는 그 집에 **페인트를 칠하고 조명을 다는 시간**입니다. 기능은 그대로 두고, 겉모습만 내가 원하는 대로 바꿔서 "이건 진짜 내 게임이다"라는 느낌을 만들어 봅니다.

---

## 따라 하기

### 단계 ① 전체 색깔 바꾸기

AI에게 화면 전체 색 분위기를 부탁해 봅니다.

!!! quote "🗣️ 이대로 복사해서 붙여넣으세요 (AI에게 하는 말)"
    ```
    게임 전체 색깔을 청록색과 검은색이 어우러진
    네온 느낌으로 바꿔줘.
    방향키 4개(← ↓ ↑ →)도 각각 다른 색으로 보이게 해줘.
    예를 들면 왼쪽은 핑크, 아래는 파랑,
    위는 초록, 오른쪽은 노랑처럼.
    ```

AI가 색을 바꾸고 나면 실행해서 확인합니다.

### 단계 ② 제목과 소개 문구 바꾸기

이제 이 게임의 이름을 지어 줍니다. 원하는 문구를 그대로 프롬프트에 적으면, AI가 화면에 그대로 반영해 줍니다.

!!! quote "🗣️ 이대로 복사해서 붙여넣으세요 (AI에게 하는 말)"
    ```
    시작화면 제목을 "리듬 스텝"으로 바꿔줘.
    제목 아래 작은 설명 문구로
    "네 방향키로 비트를 밟는 DDR 스타일 리듬 게임"을 넣어줘.
    ```

위 **그림 16.1**을 다시 보면, 제목과 설명 문구가 바뀐 것을 확인할 수 있습니다. 곡 이름(예: 기본 곡·120 BPM)이나 조작법 안내 문구도 같은 방식으로 원하는 말로 바꿔 달라고 부탁하면 됩니다.

!!! tip "💡 이럴 땐 — 문구를 여러 개 한꺼번에 바꾸고 싶다면"
    한 번에 하나씩 부탁해도 되고, "제목은 이거, 설명은 이거, 버튼 글자는 이거로 바꿔줘"처럼 **한 프롬프트에 몰아서** 부탁해도 됩니다. AI가 한 번에 다 바꿔 줍니다.

### 단계 ③ 글꼴(폰트) 바꿔 보기

글자 모양도 프롬프트 한 줄로 바꿀 수 있습니다.

!!! quote "🗣️ 이대로 복사해서 붙여넣으세요 (AI에게 하는 말)"
    ```
    제목 글꼴을 지금보다 더 굵고 눈에 띄게 바꿔줘.
    ```

### 단계 ④ 맞췄을 때 반짝이는 효과 더하기

이번엔 눈에 재미를 더해 봅니다. 노트를 딱 맞게 맞췄을 때 화면이 살짝 반짝이는 효과입니다.

!!! quote "🗣️ 이대로 복사해서 붙여넣으세요 (AI에게 하는 말)"
    ```
    노트를 딱 맞게(PERFECT) 눌렀을 때
    화면이 살짝 반짝이거나 색이 번쩍이는 효과를 넣어줘.
    콤보가 10을 넘으면 화면 테두리에
    은은하게 색이 도는 효과도 추가해줘.
    ```

### 단계 ⑤ 마음에 들 때까지 다시 부탁하기

한 번에 딱 마음에 드는 결과가 안 나올 수 있습니다. 그럴 땐 계속 이어서 말하면 됩니다.

!!! quote "🗣️ 이렇게 이어서 말해도 됩니다"
    ```
    색이 너무 화려해. 좀 더 차분하게 해줘.
    (또는) 이 파란색 말고 보라색 계열로 바꿔줘.
    (또는) 반짝이는 효과가 너무 약해. 더 확실하게 보이게 해줘.
    ```

!!! warning "⚠️ 조심 — 너무 화려하면 오히려 게임하기 불편해집니다"
    배경이나 효과가 너무 요란하면 노트와 판정선이 잘 안 보여서 오히려 플레이가 힘들어질 수 있습니다. 꾸미고 나서 실제로 몇 번 플레이해 보고, 노트가 잘 보이는지 꼭 확인하세요. 안 보이면 "배경은 더 차분하게, 노트만 눈에 잘 띄게 해줘"라고 다시 부탁하면 됩니다.

---

!!! success "✅ 여기까지 됐으면"
    - ☐ 게임 전체 색깔(과 방향키 색)을 내 마음대로 바꿔 봤다
    - ☐ 제목·설명 문구를 내가 원하는 말로 바꿔 봤다
    - ☐ 맞췄을 때 반짝이는 효과를 넣고, 노트가 잘 보이는지 확인했다

!!! abstract "📌 핵심 요약"
    - 게임의 **기능은 그대로 두고 겉모습만** 프롬프트로 바꾸는 것이 이번 차시입니다.
    - 색·글꼴·문구·효과 모두 "이렇게 바꿔줘"라고 자연스럽게 부탁하면 됩니다.
    - 결과가 마음에 안 들면 "더 화려하게", "더 차분하게"처럼 **계속 이어서** 다시 부탁하면 됩니다.
    - 꾸민 뒤엔 반드시 실제로 플레이해 보고 노트가 잘 보이는지 확인합니다.
