## 13차시 · 키를 눌러 노트 치기


```
지금 화면에서 노트가 판정선에 왔을 때 방향키를 누르면
그 노트가 사라지도록 만들어줘.

- 방향키를 누른 순간과 노트가 판정선을 지나가는 순간이
  얼마나 가까운지 계산해서
- 아주 정확하면 "PERFECT"
- 조금 빠르거나 늦으면 "GREAT"
- 많이 빗나가면 "GOOD"
- 아예 못 맞추고 지나가면 "MISS"
라고 화면 가운데에 크게 글씨로 보여줘.

판정마다 색깔도 다르게 해줘. 예를 들면 PERFECT는 초록,
MISS는 빨간색처럼 한눈에 구분되게.
```


## 14차시 · 점수·콤보 보여주기


```
노트를 PERFECT나 GREAT, GOOD으로 맞출 때마다
점수(SCORE)가 올라가게 해줘. PERFECT가 가장 점수를
많이 주면 좋겠어.

그리고 콤보(COMBO, 연속으로 성공한 횟수)도 만들어줘.
잘 맞출 때마다 콤보가 1씩 늘어나고, MISS가 나오면
콤보가 다시 0으로 끊기게 해줘.

화면 오른쪽 위에 지금까지 모은 점수와, 지금 이어지고
있는 콤보 숫자를 크고 잘 보이게 보여줘. 그리고 지금까지
얼마나 정확하게 쳤는지 정확도(%)와 등급(예: S, A, B…)도
같이 보여줘.
```


```
점수가 올라갈 때 숫자가 통통 튀듯이
살짝 커졌다가 작아지는 효과를 넣어줘.

그리고 콤보가 10을 넘으면 콤보 숫자 색깔이
눈에 확 띄게 바뀌도록 해줘.
```


## 15차시 · 음악·박자와 노트 맞추기


```
게임 시작 전에 난이도를 고르는 화면을 만들어줘.
EASY, NORMAL, HARD, EXPERT 4단계로 나누고
(난이도가 높을수록 노트가 더 촘촘하게 나오면 좋겠어).
그리고 노트가 떨어지는 속도를 조절하는 슬라이더랑,
연습할 때 노래를 0.5배, 0.75배로 느리게 들을 수 있는
버튼도 같이 넣어줘.
```


```
내가 화면을 보고 정확히 맞춰 누르는데도
자꾸 판정이 늦거나 빠르게 나와.
내가 직접 판정 시점을 몇 십 밀리초(ms) 단위로
당기거나 늦출 수 있는 "타이밍 보정" 기능을 만들어줘.
+와 - 버튼으로 조절하고, 지금 보정값이 몇 ms인지
화면에 숫자로 보여줘.
```

```

네온 느낌으로 바꿔줘.
방향키 4개(← ↓ ↑ →)도 각각 다른 색으로 보이게 해줘.
예를 들면 왼쪽은 핑크, 아래는 파랑,
위는 초록, 오른쪽은 노랑처럼.
```

```
"네 방향키로 비트를 밟는 DDR 스타일 리듬 게임"을 넣어줘.
```


```
제목 글꼴을 지금보다 더 굵고 눈에 띄게 바꿔줘.
```

```
화면이 살짝 반짝이거나 색이 번쩍이는 효과를 넣어줘.
콤보가 10을 넘으면 화면 테두리에
은은하게 색이 도는 효과도 추가해줘.
```

```
(또는) 이 파란색 말고 보라색 계열로 바꿔줘.
(또는) 반짝이는 효과가 너무 약해. 더 확실하게 보이게 해줘.
```
