# FORMAT — 새 교재(v2) 집필 규격 (시니어·왕초보용)

기존 교재보다 **더 크게, 더 천천히, 더 그림 많이.** 독자는 컴퓨터가 낯선 60~70대도 포함. 한 화면에 한 동작.

## 0. 대전제
- **코드를 보여주지 않는다.** 본문에 등장하는 "박스에 적힌 글"은 오직 **① AI에게 넣는 프롬프트**와 **② 화면에 그대로 뜨는 안내문**뿐. `.py`, `.js`, `.ino` 소스는 싣지 않는다.
- **한 번에 한 걸음.** 모든 조작은 "① 이 버튼을 누른다 → (그림) → ② 이렇게 뜬다" 식으로 쪼갠다.
- **그림이 본문보다 먼저 말한다.** 각 단계마다 스크린샷/사진을 붙이고, 글은 그림을 설명하는 보조로 쓴다.
- **존댓말·짧은 문장.** 한 문장 한 동작. 전문용어는 처음 나올 때 괄호로 즉시 풀어 준다.

## 1. 조직 단위
```
마당(부)  →  차시(레슨)  →  단계(스텝)
```
- **마당** = 큰 묶음(예: 1마당 "컴퓨터랑 인사하기"). 로마자·색으로 구분.
- **차시** = 한 번에 앉아서 끝내는 분량(30~50분). 제목은 "무엇을 하게 되는지"로.
- **단계** = 번호 붙은 실제 조작(①②③…). 단계마다 그림 1장 원칙.

## 2. 차시 템플릿
```markdown
# N차시 · 제목

!!! note "이번 차시에 하는 일"
    - (한 일 1) — 손에 남는 결과로 적기
    - (한 일 2)
    - (한 일 3)

> ⏱️ 걸리는 시간: 약 OO분 · 🧰 준비물: (있으면)

---

## 왜 이걸 하나요? (2~4문장, 그림 1장)
...쉬운 비유로 동기 부여...

## 따라 하기

### 단계 ① (동사로 시작: "…를 켭니다")
(그림)
한 문장 설명 + "이렇게 보이면 성공입니다" 확인 문구.

### 단계 ② …

---

!!! tip "💡 이럴 땐"
    자주 나오는 갈림길·요령.

!!! warning "⚠️ 조심"
    실수하기 쉬운 지점 1개.

!!! success "✅ 여기까지 됐으면"
    이번 차시 성공 기준 체크리스트(☐ 3개 내외).

!!! question "🤔 혼자 해보기"
    Q. …?  ✍️ ______________________

!!! info "🔎 낱말 사전"
    - **용어** — 한 줄 뜻.

> **다음 차시 예고** — 한 문장.
```

## 3. 프롬프트 박스 (이 책의 심장)
독자가 AI에게 **그대로 복사해 붙여넣는** 지시문. 항상 아래 형태로, 눈에 확 띄게.
```markdown
!!! quote "🗣️ 이대로 복사해서 붙여넣으세요 (AI에게 하는 말)"
    ```
    (프롬프트 본문 — 자연스러운 한국어. 존댓말도 반말도 OK, 핵심은 '뭘 원하는지')
    ```
```
- 프롬프트는 **말하듯 자연스럽게.** "프롬프트 엔지니어링" 없이도 된다는 걸 몸소 보여준다.
- 프롬프트 뒤엔 항상 **"이렇게 답하면 정상입니다"** 예시 화면(그림)을 붙인다 → 결과 보장.

## 4. 그림 규약 (기존 시스템 계승)
- 인라인 마커:
  `<!-- FIG: id=cNN-fXX | type=스크린샷|사진|일러스트|표 | src=capture|blog|web|manual | file=images/cNN/cNN-fXX.png -->`
- 마커 아래 캡션 블록:
  ```
  > **그림 N.X — 캡션**
  >
  > *[캡처 지시 또는 출처: …]*
  ```
- `src` 값: `capture`(내가 직접 화면 열어 캡처) · `blog`(한국 블로그 이미지, 출처 표기) · `web`(웹검색 이미지) · `manual`(사람이 촬영).
- 파일은 `book2/images/cNN/` 아래. 매니페스트는 `book2/images.yaml`(선택).
- **차시당 그림 최소 6장** 목표(단계 수만큼). 왕초보라 화면 캡처가 곧 설명.

## 5. 톤/표기
- 그림 번호: 점 표기 `그림 1.3`, `표 2.1`.
- 도구 이름 표기: Claude Code / Codex / OpenCode / Antigravity (영문 그대로), 첫 등장 시 "(AI에게 코딩 시키는 도구)" 부연.
- 클릭 대상은 **굵게**: **[다운로드]** 버튼을 누릅니다.
- 키 입력은 코드폰트: `node -v` 처럼.
- 금지: 영어 전문용어 남발, 긴 배경 이론, "간단히", "쉽게"라고만 말하고 넘어가기.
