# YLBooks

바이브코딩 교재 집필 저장소.

## 📘 말로 만드는 내 키링 리듬게임 — `book2/`

컴퓨터가 낯선 완전 초보(60~70대도)도 **AI에게 말(프롬프트)만 걸어서** 리듬게임을 만들고, **키링 키보드**(블루투스·방향키)로 직접 플레이하도록 안내하는 교재입니다. 본문에 소스코드는 싣지 않고, **실제 화면 캡처**로 한 단계씩 따라가게 합니다.

- **구성**: 서문 + 21차시 + 부록 A~D (6마당)
  - 1 바이브코딩 첫 만남 · 2 컴퓨터 기초(터미널·탐색기·Node) · 3 AI 도구 설치(Claude Code·Codex·OpenCode·Antigravity) · 4 리듬게임 만들기(Expo Web) · 5 키링으로 플레이 · 6 자랑·로드맵
- **완성본(읽기용)**: `book2/리듬게임교재_전체_v3.docx`
- **원고(집필 소스)**: `book2/lessons/*.md`
- **그림**: `book2/images/`
- **기획·규격**: `book2/PLAN.md`(전체 기획), `book2/FORMAT.md`(집필 규격), `book2/CONTEXT.md`(확정 사양·용어), `book2/CAPTURE_QUEUE.md`(그림 현황·저자 촬영 목록)
- **캡처 도구**: `book2/tools/`(창 단위 캡처 등)

> 상태: 초안 완성. 일부 그림(설치 마법사·블루투스 페어링·물리 키링 사진 등)은 저자 촬영 대기(`CAPTURE_QUEUE.md` 참조).
